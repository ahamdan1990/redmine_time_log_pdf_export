<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>Install <code>additionals</code> plugin for <a href="https://www.redmine.org" rel="nofollow">Redmine</a>.</p>
<div class="highlight highlight-source-shell"><pre>$ <span class="pl-c1">cd</span> <span class="pl-smi">$REDMINE_ROOT</span>
$ git clone -b stable https://github.com/AlphaNodes/additionals.git plugins/additionals
$ bundle install --without development <span class="pl-c1">test</span>
$ bundle <span class="pl-c1">exec</span> rake redmine:plugins:migrate RAILS_ENV=production
$
$ <span class="pl-c"><span class="pl-c">#</span> if you want to use smiley/emoji legacy support, you have to put emoji icons to</span>
$ <span class="pl-c"><span class="pl-c">#</span> $REDMINE_ROOT/public/images/emoji</span>
$ <span class="pl-c"><span class="pl-c">#</span> To obtain image files, run the gemoji extract command on macOS Sierra or later:</span>
$ bundle <span class="pl-c1">exec</span> gemoji extract public/images/emoji
$
$ <span class="pl-c"><span class="pl-c">#</span> if you to not have macOS, you can put these files manually to $REDMINE_ROOT/public/images/emoji</span>
$ <span class="pl-c"><span class="pl-c">#</span> see https://github.com/github/gemoji for more infos</span></pre></div>
<p>Restart your application server (apache with passenger, nginx with passenger, unicorn, puma, etc.) and <code>Additionals</code> is ready to use.</p>
<p>More information about installation of Redmine plugins, you can find in the official <a href="https://www.redmine.org/projects/redmine/wiki/Plugins" rel="nofollow">Redmine plugin documentation</a>.</p>

<p>Update <code>additionals</code> plugin for <a href="https://www.redmine.org" rel="nofollow">Redmine</a>.</p>
<div class="highlight highlight-source-shell"><pre>$ <span class="pl-c1">cd</span> <span class="pl-smi">$REDMINE_ROOT</span>/plugins/additionals
$ git pull
$ <span class="pl-c1">cd</span> ../..
$ bundle install --without development <span class="pl-c1">test</span>
$ bundle <span class="pl-c1">exec</span> rake redmine:plugins:migrate RAILS_ENV=production</pre></div>
<p>Restart your application server (apache with passenger, nginx with passenger, unicorn, puma, etc.) and <code>Additionals</code> is ready to use.</p>

<h2><a id="user-content-developer-features" class="anchor" aria-hidden="true" href="#developer-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Developer features</h2>
<p>As Redmine does not support asset pipeline, we need to install Javascript plugins as Redmine plugins to load them globally.</p>
<p>If no common files are used as library plugin, every plugin has to deliver the same files. And if more
than one version of a library is delivered with each Redmine plugin, there is a problem.</p>
<p>Therefore if developer uses this plugin for the provided libraries, everything works smoothly.
Don't worry, if you only need a subset of the provided libraries. If you do not load a library, it is not used.</p>
<p>It provides :</p>
<ul>
<li><a href="https://www.chartjs.org/" rel="nofollow">Chart.js 2.9.4</a></li>
<li><a href="https://github.com/nagix/chartjs-plugin-colorschemes">Chart.js Plugin colorschemes 0.4.0</a></li>
<li><a href="https://github.com/chartjs/chartjs-plugin-datalabels">Chart.js Plugin datalabels 0.7.0</a></li>
<li><a href="https://clipboardjs.com/" rel="nofollow">clipboardJS 2.0.6</a></li>
<li><a href="https://d3js.org/" rel="nofollow">d3 6.3.1</a></li>
<li><a href="https://d3plus.org/" rel="nofollow">d3plus v2.0.0-alpha.30</a></li>
<li><a href="https://fontawesome.com/" rel="nofollow">FontAwesome 5.15.1</a></li>
<li><a href="https://github.com/knsv/mermaid/">mermaid 8.8.4</a></li>
<li><a href="https://select2.org/" rel="nofollow">Select2 4.0.13</a></li>
</ul>
<p>And a set of various Rails helper methods (see below).</p>
